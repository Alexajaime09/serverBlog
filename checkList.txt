==============// Code Status // ==================


200 [ok] => request successful / Requete reussie / solicitud exitosa


201 [created] cuando se crea un documento => resource created  / ressource creee / recurso creado


400 [bad request] datos incorrectos o incompletos => invalid request / requete invalide / solicitud invalida


401 [Unauthorized]=> usuario no autenticado => authentication required / authentification requise / autenticacion requerida


403 [Forbidden] no se tiene permiso => access forbiden / acces interdit / acceso prohibido

404 [not found] Id o recurso inexistente => resource not found / ressource introuvable / recurso no encontrado

409 [conflict] datos duplicados o incompatibles =>
conflict detected / conflit detecte / conflicto detectado

422 Datos válidos en formato, pero lógicamente incorrectos

500 [internal server error] error inesperado en el backend => internal server error / erreur interne du serveur / error interno del servidor
